# UI House Style (Visual Kit) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Reskin all 5 trivia screens to the Visual Kit / Gumroad house style, wired to the visual-kit token pipeline so palette swaps propagate with no markup edits.

**Architecture:** `theme.css` (visual-kit's generated `--visual-kit-*` tokens) is the source of truth, linked before `app.css` on every page. `app.css` is a thin adapter: its `:root` aliases the app's legacy var names to those tokens; all component rules already read the legacy aliases, so they reskin through them. No markup migration, no JS change.

**Tech Stack:** Static HTML/CSS, FastAPI backend (already built), agent-browser for visual verification.

## Global Constraints

- Branch: work on `ui-visual-kit-house-style` (already created). Never commit to `main`.
- No em dashes in any written content (visual-kit convention).
- Never hand-edit `static/theme.css` — it is copied from visual-kit's generated output.
- No JS, API, or game-logic changes.
- Page JS keys off element IDs and `.hidden` only — preserve every `id=` and class name in markup.

- [ ] **Step 1: Write the demo seeder**

Create `scripts/seed_demo.py`:

```python
"""Seed demo.db with teams, a round of questions, scores, and a submission so
the data-heavy screens can be visually verified. Usage:
    python scripts/seed_demo.py <phase>
where <phase> is one of: lobby round_open reveal marking final_wager tiebreak
Re-runnable: rounds/teams/questions/marks are inserted once, phase is updated
every run."""
import sys
from app.main import create_app
from app import db as dbmod

PHASE = sys.argv[1] if len(sys.argv) > 1 else "round_open"

create_app("demo.db")  # ensures schema + categories + game row (code, host_key)
conn = dbmod.connect("demo.db", check_same_thread=True)

cat = conn.execute("SELECT id FROM category ORDER BY display_order LIMIT 1").fetchone()["id"]

if not conn.execute("SELECT 1 FROM round WHERE id=1").fetchone():
    conn.execute("INSERT INTO round (id,title,category_id,display_order,is_final) "
                 "VALUES (1,'Round One: General Knowledge',?,1,0)", (cat,))
    qs = [
        "What is the capital of Australia?",
        "Which element has the chemical symbol Au?",
        "In what year did the Berlin Wall fall?",
        "Who painted the ceiling of the Sistine Chapel?",
        "What is the largest planet in our solar system?",
    ]
    for i, text in enumerate(qs):
        conn.execute("INSERT INTO question (author_name,category_id,text,answer,round_id,display_order,point_value) "
                     "VALUES ('demo',?,?,'',1,?,1)", (cat, text, i))

teams = [("Sandy Bottoms", 5), ("The Clam Diggers", 3), ("Beach Please", 4),
         ("Wave Runners", 2), ("Low Tide Lushes", 1)]
for name, score in teams:
    row = conn.execute("SELECT id FROM team WHERE name_lower=?", (name.lower(),)).fetchone()
    if row:
        tid = row["id"]
    else:
        cur = conn.execute("INSERT INTO team (name,name_lower,recovery_code) VALUES (?,?,?)",
                           (name, name.lower(), f"DEMO{score:04d}"))
        tid = cur.lastrowid
    # one mark per question giving this team `score` correct answers
    for q in conn.execute("SELECT id FROM question WHERE round_id=1").fetchall():
        conn.execute("INSERT OR IGNORE INTO mark (team_id,question_id,is_correct,score) VALUES (?,?,?,?)",
                     (tid, q["id"], 1 if score > 0 else 0, 1.0 if score > 0 else 0.0))
        score -= 1

# a submission for team 1 so the host marking panel has a row
t1 = conn.execute("SELECT id FROM team WHERE name_lower='sandy bottoms'").fetchone()["id"]
conn.execute("INSERT OR IGNORE INTO submission (team_id,round_id,photo_path) VALUES (?,1,'demo.jpg')", (t1,))

crid = 1 if PHASE in ("round_open", "final_open", "marking", "final_wager") else None
conn.execute("UPDATE game SET phase=?, current_round_id=? WHERE id=1", (PHASE, crid))
conn.commit()
hk = conn.execute("SELECT host_key FROM game WHERE id=1").fetchone()["host_key"]
print(f"seeded phase={PHASE}  host_key={hk}")
```
